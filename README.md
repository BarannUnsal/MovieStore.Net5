# MovieStore Web APP

Uygulama içerisindeki özellikler
1. Müşteriler
- Müşteri ekleme
- Müşteri güncelleme
- Müşteri listeleme
- Müşteri silme

2. Aktörler
- Aktör ekleme
- Aktör güncelleme
- Aktör listeleme
- Aktör silme

3. Yönetmenler
- Yönetmen ekleme
- Yönetmen güncelleme
- Yönetmen listeleme
- Yönetmen silme

4. Filmler
- Film ekleme
- Film güncelleme
- Film listeleme
- Film silme
